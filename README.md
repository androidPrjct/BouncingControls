# BouncingMenu
一款弹跳菜单插件
>**弹跳菜单控件件**是一款专门脱离于Activity的一个自定义控件。特点概述：
 
- **动态加载** ：不用再XML布局文件中生命定义；
- **框架化** ：可以直接像*Toast.makeText(getBaseContext(),"",Toast.LENGTH_SHORT).show()*，利用了建造者模式来调用；
- **动画美感** ：利用了属性动画。
